features

More than a signal bar

CoreWLAN under the hood, a clean SwiftUI popover on top. Everything you need to understand — and fix — a flaky connection.

📊

Live signal metrics

RSSI, latency, link speed and SNR with animated bars, updated event-driven via CoreWLAN.

📱

Hotspot detection

Spots an iPhone Personal Hotspot automatically via NWPathMonitor — and skips the speed test to save your data.

⚡

Download speed test

Optional real throughput measurement when you want the full picture.

🔎

Network scanner

Lists nearby networks grouped by SSID with an AP-count badge; join directly with a password prompt.

📍

Location auto-switch

Capture a location and auto-switch to its preferred SSID on arrival — opt-in, via CoreLocation.

🔄

Auto-reconnect

Retries the connection when dropped, at a configurable interval. Toggle WiFi from the popover.

🔔

Smart notifications

Optional alerts on connection loss and when you switch onto a hotspot — never silent.

🚀

Launch at login & auto-update

SMAppService start-up plus Sparkle 2 update prompts — you're always in control.

🟢

Color-coded icon

Green / orange / red menu bar icon with a searching-pulse animation. Hide the native macOS WiFi icon and use this instead.

per-use scoring

Is it good enough for this?

The same connection can be perfect for email and useless for a video call. WifiManager rates your link for each use case, so you know before it fails.
